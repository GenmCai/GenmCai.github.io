大学物理公式和名词整理

大学物理公式

描述质点运动的物理量

位矢

公式: $\vec{r} = \vec{r}(t)$

位置矢量是描述某一时刻质点所在空间位置的物理量。

位移

公式: $\Delta{\vec{r}} = \vec{r_2} - \vec{r_1}$

位移是描述质点位置变化的物理量。

速度

公式:

  1. 平均速度 $\displaystyle \bar{v} = \frac{\Delta{s}}{\Delta{t}}$

  2. 速度 $\displaystyle v = \lim_{\Delta{t}\rightarrow0}\frac{\Delta{\vec{r}}}{\Delta{t}} = \frac{dr}{dt}$

速度是描述质点运动方向和运动快慢的物理量。

加速度

公式:

  1. 平均加速度 $\displaystyle \bar{\vec{a}} = \frac{\Delta{\vec{v}}}{\Delta{t}}$

  2. 瞬时加速度，简称加速度$\displaystyle a = \lim_{\Delta{t}\rightarrow0}\frac{\Delta{\vec{v}}}{\Delta{t}} = \frac{d\vec{v}}{dt} = \frac{d^2\vec{r}}{dt^2}$

加速度是描述质点速度方向和速度大小变化快慢的物理量。

圆周运动的角量描述

角位置

公式: $\theta = \theta(t)$

一质点在$O_{xy}$平面内，绕圆心$O$做圆周运动，在某一时刻$t$位于点A，半径$OA$与坐标轴$O_x$的夹角的为$\theta$,角$\theta$称为角位置，其值随时间t而变化，是时间t的函数，即$\theta = \theta(t)$

PS: 式$\theta = \theta(t)$称为质点做圆周运动时的运动方程。

在国际单位制中，角位置的单位为$rad$(弧度)。

角位移

表示: $\Delta{\theta}$

设在时刻$t + \Delta{t}$，质点到达点$B$，半径$OB$与$O_x$的夹角为$\theta +\Delta{\theta}$。在时间$\Delta{t}$内，质点转过的角度$\Delta{\theta}$称为质点的角位移。角位移不仅有大小而且有方向。一般规定质点沿逆时针方向转动时角位移取正值，沿顺时针方向转动时角位移取负值。

在国际单位制中，角位移的单位为$rad$(弧度)。

角速度

公式: $\displaystyle \omega = \lim_{\Delta{t}\rightarrow0}\frac{\Delta{\theta}}{\Delta{t}} = \frac{d\theta}{dt}$

角位移$\Delta{\theta}$与对应时间$\Delta{t}$之比，称为在$\Delta{t}$时间内质点对点$O$的平均角角速度，平均角加速度的极限值称为瞬时角加速度，简称角加速度，用$\omega$表示，即$\omega = \lim_{\Delta{t}\rightarrow0}\frac{\Delta{\theta}}{\Delta{t}} = \frac{d\theta}{dt}$。

在国际单位制中，角速度的单位为$rad/s$(弧度每秒)。

角加速度

公式: $\displaystyle \alpha = \frac{d\omega}{dt} = \lim_{\Delta{t}\rightarrow0}\frac{\Delta{\omega}}{\Delta{t}} = \frac{d^2\theta}{dt^2}$

牛顿运动定律及其应用

1. 牛顿第一定律

任何物体都保持静止或匀速直线运动状态，直到外力迫使它改变这种状态为止，即$F = 0$时，有$v = 常矢量$。

2. 牛顿第二定律

物体所获得的加速度的大小与其所受合外力的大小成正比，与物体的质量成反比，加速度的方向与合外力的方向相同，即 $\displaystyle F = m\vec{a} = m \frac{d\vec{v}}{dt}$。

在国际单位制中，力$F$的单位为$N$(牛顿)。

3. 牛顿第三定律

当物体A以力$F$作用在物体B上时，物体B必定同时以力$F’$作用在物体A上，$F$和$F’$在一条直线上，大小相等，方向相反，即$F = -F’$。

PS: 牛顿第三定律中的作用力与反作用力是指存在相互作用的两个物体受的力之间的联系，作用力与放反作用力是分别作用在不同物体上的，尽管它们的数值相等，方向相反，但并不抵消，它们将同时存在或同时消失，而且是同一性质的力，如作用力是摩擦力，反作用力也一定是摩擦力，绝不可能是其他性质的力。

万有引力

公式: $\displaystyle F = -G_0\frac{m_1m_2}{r^2}\vec{e_r}$

重力

公式: $G = m\vec{g}$

守恒定理

机械能

机械能 = 动能 + 势能

功

公式:

  1. 元功 $\displaystyle dA = \vec{F} \cdot d\vec{r} = F\cos{\theta}|d\vec{r}|$

  2. 做功 $\displaystyle A = Fs\cos{\theta} = \int^{B}_{A}{F\cos{\theta}ds}$

在国际的单位制中，功的单位为$J$(焦耳)。

动量

公式: $\displaystyle p = mv$

动量（Momentum）又称线性动量（Linear Momentum）。在经典力学中，动量（是指国际单位制中的单位为kg·m/s ，量纲MLT⁻¹）表示为物体的质量和速度的乘积，是与物体的质量和速度相关的物理量，指的是运动物体的作用效果。动量也是矢量，它的方向与速度的方向相同。

功率

公式: 功率 $\displaystyle P = \lim_{\Delta{t}\rightarrow0}\frac{\Delta{A}}{\Delta{t}} = \frac{dA}{dt}$

功率是表征做功快慢的物理量。功率的定义是力在单位时间内所做的功，如果在$\Delta{t}$内，力对物体做功$\Delta{A}$，则$\bar{p} = \frac{\Delta{A}}{\Delta{t}}$。

动能

公式: $\displaystyle E_k = \frac{1}{2}mv^2$

定义: 物体由于运动而具有的能量，称为物体的动能。它的大小定义为物体质量与速度平方乘积的二分之一。

势能

表示: $E_P$

在物体从点A移动到点B的过程中，保守力做功总是等于某个位置函数之差，而功总是与能量变化相联系，因此，这个位置函数也应该具有能量的意义，将其称为势能函数，简称势能，以$E_p$表示，即$A = \int^B_A{F \cdot d\vec{r} = E_{pA} - E_{pB}}$。

质点的动能定理

物体从点A到点B，合外力对物体所作的功为

$\displaystyle A=\int^{B}_{A}{F\cos{\theta}ds}=\int^{v_2}_{v_1}{mvdv}=\frac{1}{2}mv_2^2-\frac{1}{2}mv_1^2$

$A = E_{k2} - E_{k1}$

合外力对物体所做的功等于物体动能的增量，这一结论称为质点的动能定理。

质点系的动能定理

由互相作用的若干个物体组成的系统，一般简称为质点系。系统内各物体间的相互作用力称为内力，系统之外的其他物体对系统内任意一物体的作用称为外力。

设作用在第$i$个物体上的力对它做的功为$A_i$，由质点的动能定理可得

$\displaystyle A_i = \frac{1}{2}m_{i}v_{i2}^2 - \frac{1}{2}m_{i}v_{i1}^2$

对系统内各个物体都应用动能定理，并相加得到

$\sum_{i=1}^{n}{A_i} = \sum_{i=1}^{n}{E_{ki2} - E_{ki1}} = \sum_{i=1}^{n}{\frac{1}{2}m_{i}v_{i2}^2} - \sum_{i=1}^{n}{\frac{1}{2}m_{i}v_{i1}^2}$

这些力中既有该系统的内力，也有该系统的外力。

$A_内+A_外 = \sum_{i=1}^{n}{E_{ki2} - E_{ki1}}$

所有外力和内力对质点系所做功的总和等于系统动能的增量，这一结论称为质点系的动能定理

$$ \begin{cases} A_外 + A_{保内} + A_{非保内} = \sum^n_{i=1}{E_{ki2} - \sum^n_{i=1}E_{ki1}}\\ A_{保内} = \sum^n_{i=1}{E_{pi2} - \sum^n_{i=1}E_{pi1}}\\ A_外 + A_{非保内} =(\sum^n_{i=1}{E_{ki2} + \sum^n_{i=1}{E_{pi2}) - (\sum^n_{i=1}E_{ki1}} + \sum^n_{i=1}E_{pi1}})\\ A_外 + A_{非保内} = E_2 - E_1 \end{cases}$$

外力和非保守内力所作功的总和等于系统机械能的增量，这一结论称为质点系的动能定理。

机械能守恒定律

$$ \displaystyle \begin{cases} A_外 + A_{非保内} = 0\\ E_1 = E_2\\ \sum^n_{i=1}{E_{pi1} + \sum^n_{i=1}{E_{ki1} = \sum^n_{i=1}E_{pi2}} + \sum^n_{i=1}E_{ki2}} \end{cases}$$

当作用在系统上的外力和非保守内力不做功时，或者它们所作的总功为零时，系统内物体的动能和势能可以互相转换，但它们的总和，即系统的机械能保持不变，这一结论称为机械能守恒定律。

能量守恒定律

对一个孤立系统(和外界即没有物质交换，也没有能量交换的系统)来说，不论发生任何变化，各种形式的能量都可以互相转化，但它们的总和是一个常量，这一结论称为能量守恒与转化定律，简称能量守恒定律。
